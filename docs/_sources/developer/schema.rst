*****************
Validation Schema
*****************

C-PAC uses `voluptuous <alecthomas.github.io/voluptuous/docs/_build/html>`_ to validate pipeline ~:py:class:`CPAC.utils.configuration.configuration.Configuration`s against defined types and value restrictions. Default values should be set in :doc:`the default pipeline </user/pipelines/default>` along with comments explaining each option.
